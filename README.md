# DocMaster AI (Upgraded)

DocMaster AI is a lightweight, mobile-friendly document productivity platform.

## Features
- PDF to Word Conversion
- OCR Scan (Arabic & English)
- PDF Merging/Splitting
- AI Text Enhancement (Gemini)

## Deployment
This space is configured to run via Docker on Hugging Face Spaces.
